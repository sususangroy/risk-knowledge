## 一、市场概况

墨西哥是拉丁美洲第二大经济体，约 5,000 万成年人没有银行账户。"免息分期（Meses Sin Intereses）"的信贷文化在墨西哥根深蒂固，早于互联网时代就普遍存在于零售业，为 BNPL 产品提供了天然的用户认知基础。

### 关键量化数据

| 指标 | 数据 |
|------|------|
| 无银行账户成年人 | 约 **5,000 万**（约占成年人口 50%） |
| 正规信贷渗透率 | 约 35%–40% |
| 人均获客成本（CAC） | 约 70–200 美元（约 500–1,420 元人民币） |
| 坏账率（90 天逾期率） | 约 13%–14% |
| 消费金融科技贷款余额（估算） | 约 40–60 亿美元 |
| 总消费信贷市场（含银行） | 约 700–800 亿美元 |
| CNBV 持牌 Fintech 机构数量 | 约 70 家（截至 2023 年底，IFC + IFPE） |

## 二、监管框架

### 监管机构

| 机构 | 主要职责 |
|------|----------|
| CNBV（国家银行和证券委员会） | 发放 IFC（众筹/P2P）和 IFPE（电子支付）牌照，监管持牌机构 |
| CONDUSEF（金融用户保护委员会） | 受理消费者投诉、发布产品比较报告、处罚不公平合同 |
| Banxico（墨西哥央行） | 支付系统、开放银行标准 |

### 核心利率规则：CAT 强制披露

墨西哥**没有全国性消费贷利率上限**，但所有信贷产品必须以 **CAT（Costo Anual Total，全年总成本，等同于 APR）** 的方式强制披露，且须在广告中显著展示。

> **CAT = 利率 + 所有费用（手续费、年费等）的年化成本合计**，是比单纯"利率"更全面的成本指标。

**CAT 高企的原因**：
1. 无利率上限，平台可合法收取极高费率
2. 高获客成本（$70–200/人）和高坏账率（13–14%）需要高利率覆盖
3. 短期小额产品（7–30 天）年化计算天然偏高

### 《金融科技法》（Ley Fintech，2018 年 3 月）

- 墨西哥是全球首批专门立法规范金融科技的国家
- 创设两类牌照：**IFC**（众筹/P2P）、**IFPE**（电子支付）
- 要求所有 Fintech 平台完成 CNBV 注册
- 执行期结束后，CNBV 从 300+ 申请者中仅批准约 **70 家**

### 近期监管动态

| 时间 | 事件 |
|------|------|
| 2022–2024 | CNBV 完成 Ley Fintech 实施期清退，大量无牌平台退出 |
| 2023 | CONDUSEF 发布专项报告，点名 20+ App 年化超 1,000% |
| 2024 | 参议院提出 APR 36% 上限草案（参考美国军事贷款法），讨论中 |
| 2023–2024 | 联邦检察院 + CNBV 联合打击"Gota a Gota"非法放贷网络 |

## 三、中国企业在墨西哥的详细布局

### 1. 滴滴出行 —— DiDi Finanzas

**背景与策略**：
- 滴滴依托其在墨西哥的网约车平台（是当地主要网约车服务商之一），以出行场景为入口切入消费金融
- 核心逻辑：司机有可验证的收入流水（接单记录），乘客有真实的消费场景，两端都是理想的信贷用户
- DiDi Finanzas 是滴滴墨西哥金融服务子品牌

**产品矩阵与利率详情**：

**① DiDi Card（滴滴信用卡）**

| 参数 | 数据 |
|------|------|
| 产品类型 | 信用卡（面向 DiDi 用户和司机） |
| 年费 | **终身免年费** |
| 核心卖点 | 高额返现、申请门槛低（低信用分也可申请） |
| CAT（全年总成本） | 通常在 **120%–130%**（不含增值税/IVA） |
| 含增值税后实际 CAT | 约 140%–150% |
| 信用额度 | 面向低信用评分人群，初始额度通常较低 |

> 墨西哥信用卡 CAT 120–130% 在国际上属于极高水平，但在墨西哥无银行账户人群市场中属于正常区间——对比银行正规信用卡 CAT 约 50%–100%，DiDi Card 定位明确的次优级客群（Near-Prime）。

**② DiDi 贷款（循环现金贷）**

| 参数 | 数据 |
|------|------|
| 产品类型 | 循环信贷额度（Revolving Credit Line） |
| 目标客群 | DiDi 司机（以接单收入为还款来源）+ DiDi 乘客 |
| 特点 | 按需支取，按实际用款天数计息 |
| 利率 | 未公开披露明确 CAT，与信用卡产品利率区间相近 |

**③ DiDi 先买后付（BNPL）**

| 参数 | 数据 |
|------|------|
| 产品类型 | 购物分期（先买后付） |
| 固定年利率 | **48%–300%**（根据期数和商户不同） |
| CAT（综合年化） | 参考值约 **400%**（不含增值税） |
| 含增值税后 CAT | 约 460%+ |

> DiDi BNPL 的 CAT 约 400% 看似极高，但需理解背景：BNPL 短期产品（如 1 个月）年化计算会数倍放大实际月费率；如实际月费率 20%，年化即为 240%；30% 月费率年化约 360%。这在无利率上限的墨西哥市场是合法的，CONDUSEF 要求的只是透明披露。

### 2. 瓴岳科技 —— Credmex

**母公司背景**：
- 瓴岳科技：中国金融科技 SaaS 和信贷运营服务商
- 在印尼以 Easycash 品牌运营，在墨西哥以 Credmex 品牌运营
- 两地均为持牌直接放贷模式

**Credmex 业务概况**：

| 参数 | 数据 |
|------|------|
| 产品类型 | 个人消费贷款 |
| 监管资质 | CNBV 持牌 Fintech（IFC 或 SOFIPO 类别） |
| 目标客群 | 无银行账户或信用记录有限的消费者 |
| 利率区间 | 参考墨西哥市场中等消费贷水平，CAT 100%–300% |

> 注：Credmex 的具体产品参数未完全公开，上述利率为基于墨西哥同类产品的市场参考区间。

- 通过新加坡控股主体 Yiren Vision Pte. Ltd. 管理海外业务

**墨西哥业务**：

## 五、利率上限立法对行业的潜在影响

**拟议方案**：APR 上限 **36%**（参考美国军事贷款法）

| 机构类型 | 若 36% 上限通过的影响 |
|----------|---------------------|
| DiDi Card（CAT 120%–130%） | 当前产品将不合规，需根本性调整定价 |
| 中资短期贷款 App（CAT 200%–1,200%） | 商业模式彻底不可行，将退出 |
| Mercado Crédito、Nubank | 部分产品接近上限，需压缩成本 |
| 本土 Kueski | 大部分现有产品需调整 |

> 截至 2025 年中，该草案尚未通过，但立法压力持续存在。墨西哥消费金融行业正处于监管转折点。

## 六、市场展望

| 机遇 | 挑战 |
|------|------|
| 5,000 万无银行账户成年人，信贷空白巨大 | 无利率上限导致行业声誉受损，立法压力增大 |
| "Meses Sin Intereses"文化为 BNPL 提供天然土壤 | 高 CAC（$70–200）+ 高坏账（13–14%）挤压利润 |
| 滴滴等出行平台的场景优势（可验证收入） | 高 CAC（$70–200）+ 高坏账（13–14%）挤压利润 |
| Ley Fintech 框架为合规运营提供法律确定性 | 利率上限立法若通过，现有高利率产品需全面重构 |
