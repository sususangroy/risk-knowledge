## 一、市场概况

菲律宾是东南亚手机互联网使用率最高的市场，约 70% 的成年人没有银行账户，但移动互联网普及率极高，形成了独特的"无银行账户但高度在线"的金融需求结构。

### 关键量化数据

| 指标 | 数据 |
|------|------|
| 总人口 | 约 1.1 亿 |
| 无/欠银行账户成年人比例 | 约 70%（约 5,800 万人） |
| SEC 登记放贷组合（含传统+科技） | 约 2,000–3,000 亿菲律宾比索（约 35–53 亿美元） |
| OLA 子市场估算 | 约 500–1,000 亿菲律宾比索 |
| SEC 历年撤销 OLA 牌照数量 | 超过 **2,000 张**（创东南亚单国纪录） |
| GCash 注册用户 | 约 **8,000 万**（约占菲律宾成年人口 80%） |
| 数字银行牌照数量 | 6 张（BSP 已发放） |

## 二、监管框架

### 监管机构分工

| 机构 | 监管范围 |
|------|----------|
| BSP（菲律宾央行） | 银行、数字银行、电子货币发行机构（EMIS）、支付系统 |
| SEC（证券交易委员会） | 非银行放贷公司（含全部 OLA），依据 RA 9474 |
| NPC（国家隐私委员会） | 数据隐私执法，亚太区执法最积极的机构之一 |

### 核心利率监管

**SEC 利率上限令（2019 年起）**

| 贷款类型 | 月利率上限 | 年化约 |
|----------|-----------|--------|
| 无担保消费贷 | **6%/月** | **约 72%** |
| 含所有费用 | 合并计算，总成本上限同步收紧 | — |

> 对比印尼：菲律宾的利率管控以月为单位，而印尼以日为单位。菲律宾 6%/月的上限对高利贷 App 是重要约束，但部分 OLA 在新规出台前年化超过 500%。

**数字银行牌照（BSP 通告 1133，2021 年）**

已发放 6 张数字银行牌照：Tonik、Maya Bank、GoTyme、UNObank、Land Bank 旗下 OFBank、SeaBank（Sea Group）

**数据隐私法（RA 10173，2012 年）**

- 禁止 App 在借贷流程之外访问通讯录
- NPC 已对数十家 OLA 提起行政处罚和刑事诉讼

**SEC MC 18/19（OLA 专项规范）**

- 禁止 OLA 访问用户通讯录、照片、位置（超出必要范围）
- 要求 App 内显示 SEC 注册号和实体地址

## 三、中国企业在菲律宾的详细布局

### 1. 蚂蚁集团 —— GCash / Mynt（合规出海标杆）

**持股结构**：
- Mynt（GCash 运营主体）股东：Globe Telecom（约 55%）、**蚂蚁集团（约 45%）**、Ayala Corporation（部分）
- 蚂蚁集团是菲律宾最大单一中资金融科技股权

**业务规模**：

| 指标 | 数据（2024 年） |
|------|----------------|
| 注册用户 | 约 **8,000 万**（约占菲律宾总人口 70%+） |
| 月活用户 | 约 5,000 万+ |
| 覆盖功能 | 汇款、账单支付、电商支付、保险、投资、借贷 |

**信贷产品详情**：

**GLoan（个人贷款）**

| 参数 | 数据 |
|------|------|
| 贷款上限 | 25,000 菲律宾比索（约 440 美元，约 3,200 元人民币） |
| 月利率 | 约 3%–5% |
| 年化利率（APR） | 约 **36%–60%** |
| 活跃借款人 | 估计 **500–700 万** |
| 资质要求 | GCash 账户活跃度 + 内部信用评分模型 |

**GGives（BNPL）**

| 参数 | 数据 |
|------|------|
| 功能 | 嵌入 Lazada 等电商平台的分期支付 |
| 免息期 | 部分商户 0% 利率（商户补贴） |
| 分期利率 | 约 2%–3%/月 |

**合规状况**：持 BSP 电子货币发行机构（EMIS）牌照，遵守 BSP、SEC 和 NPC 监管，零重大处罚记录。GCash 是菲律宾中资金融科技**合规路径的标杆**，被广泛视为"正确的出海方式"。

---

### 2. 信也科技（X Financial）—— JuanHand

**母公司背景**：
- 信也科技（NYSE: FINV）：国内"拍拍贷"P2P 转型后的纽交所上市公司
- JuanHand 是其菲律宾持牌放贷业务，全称 Cashalo Holdings / First Digital Finance Corporation（具体主体有调整）

**产品详情**：

| 产品 | 参数 |
|------|------|
| 产品类型 | 个人短期现金贷 |
| 日利率上限 | **0.025%/天** |
| 月利率 | 约 **9%–12%** |
| 综合年化利率（APR） | 最高约 **30%** |
| 贷款期限 | 通常 7–180 天 |

> **JuanHand 利率远低于印尼同类产品**（印尼 AdaKami 日利率 0.3%，菲律宾 JuanHand 日利率 0.025%），原因是菲律宾 SEC 的 6%/月利率上限约束更严。这也导致信也在菲律宾的盈利压力更大，模型更依赖规模效应和复借率。

**合规动态**：
- JuanHand 在菲律宾参议院 2022 年 OLA 听证中被点名，但后续配合监管整改
- 不同于菲律宾大量被直接吊销的 OLA，JuanHand 属于"整改后仍在运营"类别

---

### 3. 宜人智能（Yiren Digital）—— Creditable Lending Corporation

**母公司背景**：
- 宜人智能（NYSE: YRD）：宜信集团旗下上市公司，国内以"宜人贷"起家，后转型为综合金融科技平台
- 2022 年末开始布局菲律宾，2023 年初正式放贷

**菲律宾业务**：

| 参数 | 数据 |
|------|------|
| 持牌主体 | Creditable Lending Corporation（SEC 菲律宾注册放贷公司） |
| 产品类型 | 无担保小额循环贷款（Unsecured Small Revolving Loans） |
| 开始时间 | 2023 年初 |
| 近期表现 | 2025 年菲律宾业务持续扩张，国际业务已实现阶段性盈利 |

**下一步计划**：宜人智能已披露计划于 2025 年下半年将菲律宾模式复制至印度尼西亚，东南亚布局进一步扩大。

**持股架构**：通过新加坡中间控股层 Yiren Vision Pte. Ltd. 持有菲律宾和墨西哥业务。

> **利率参考**：受菲律宾 SEC 6%/月利率上限约束，与 JuanHand 同处合规价格带，年化约 36%–72%（估算）。

### 4. Sea Group（腾讯持股约 25%）—— SPayLater / SeaBank

**背景**：Sea Group 为新加坡上市公司（NYSE: SE），腾讯持有约 25% 股权，是东南亚消费金融市场最重要的单一平台。

> Sea 的三大核心业务：**电子娱乐 Garena**（已下行）、**电子商务 Shopee**、**数字金融服务 SeaMoney / Monee**。其中 SeaMoney 才是整个东南亚网贷市场规模最大的单一平台。

**菲律宾业务**：

| 产品 | 参数 |
|------|------|
| SPayLater（BNPL） | 嵌入 Shopee 平台，分期付款，免息或低息 |
| SeaBank（数字银行牌照） | 受 BSP 监管，提供存款和贷款产品 |
| SeaMoney 整体放贷 | Sea 集团在东南亚各国放贷余额合计数十亿美元 |

---

### 4. Akulaku（中国系，菲律宾运营）

- Akulaku 在菲律宾也有 OJK 类似的放贷牌照（SEC 认证）
- 主要做电商分期，产品与印尼市场逻辑相同
- 规模相对较小，主要聚焦中低端消费群体

## 五、市场展望

| 机遇 | 挑战 |
|------|------|
| GCash 生态覆盖 8,000 万用户，信贷渗透空间极大 | OLA 乱象历史阴影仍在，用户信任度受损 |
| 6 家数字银行正式运营，合规信贷产品增加 | SEC 执法资源有限，违规 App "打地鼠"持续 |
| NPC 积极执法，数据生态逐步净化 | 约 70% 成年人信用空白，风控成本高 |
| BSP BNPL 框架 2024 年落地，生态正规化 | 手机 App 分发渠道管控存在漏洞 |
