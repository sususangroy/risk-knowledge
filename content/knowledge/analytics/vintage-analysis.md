## 什么是 Vintage 分析

Vintage 分析（同期群分析）是信贷资产质量分析中最重要的方法之一。

**核心思路：** 将同一时期（通常按月）发放的贷款归为一个"同期群"（Vintage），追踪这批贷款随时间推移的逾期率变化。

## 为什么需要 Vintage 分析

普通的时点逾期率存在混淆问题：某月的逾期率高，可能是：
1. 最近放款质量变差（新增坏账）
2. 历史积累（存量风险暴露）
3. 季节性因素

Vintage 分析**剥离了这些混淆因素**，让你清楚地看到每一批次贷款的质量。

## 如何构建 Vintage 表

横轴：贷款存续月数（Months on Book, MOB）
纵轴：放款月份（Vintage 批次）
数值：该批次贷款在 MOB=n 时的累计逾期率

```
          MOB1   MOB2   MOB3   MOB6   MOB12
2024-01   0.5%   1.2%   2.1%   4.5%   8.2%
2024-02   0.6%   1.3%   2.3%   5.0%
2024-03   0.4%   1.0%   1.9%
2024-04   0.5%   1.1%
2024-05   0.3%
```

## 关键分析视角

### 1. 横向对比（同 MOB 不同批次）

比较同一 MOB 下不同批次的逾期率，判断**近期放款质量是否恶化**。

例：MOB3 逾期率从 2.1% → 2.3% → 1.9%，说明 2024-03 批次质量有所改善。

### 2. 纵向观察（同批次跨 MOB）

观察单个批次随时间的逾期率爬升曲线，判断**该批次是否趋于稳定**（通常 MOB12 后曲线趋于平坦）。

### 3. 预测终极损失

利用成熟批次的成熟率，预测新批次的最终坏账率：

```
终极损失预测 = 当前 MOBn 逾期率 / 历史同期 MOBn 成熟比例
```

## 实践中的注意事项

1. **分母口径**：通常用放款金额而非笔数（金额更能反映实际风险）
2. **逾期定义**：需统一 M1/M3 口径，不同报告中保持一致
3. **季节性**：年末通常逾期率偏低（节前还款），需结合业务背景解读
4. **产品口径**：不同产品（7天、90天、1年期）应分开做 Vintage，不可混合

## 常用工具

```python
import pandas as pd

# 构建 Vintage 透视表
vintage = df.pivot_table(
    values='is_bad',
    index='loan_month',    # 放款月份
    columns='mob',         # 贷款存续月数
    aggfunc='mean'
) * 100
```
